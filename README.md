## Dependencies
* ruby
* [Jekyll](https://jekyllrb.com)
* bundler

## Setup
1. While in cloned folder, run:

    bundle install

2. To start Jekyll, run:

    bundle exec jekyll serve

3. Navigate to `localhost:4000` to view the output of the site.
